# Portfólio — Marco Aurélio Souza Arriate

## Sobre o projeto

Este portfólio foi desenvolvido com React, utilizando Vite como bundler e Tailwind CSS para estilização. O projeto foi pensado como uma forma de apresentar minha trajetória, minhas habilidades técnicas e alguns trabalhos que venho desenvolvendo ao longo da faculdade e dos cursos complementares.

Durante o desenvolvimento, o foco principal foi aprender melhor a organização de um projeto React, o uso de componentes, rotas e a criação de uma interface moderna e responsiva.

---

## Tecnologias utilizadas

- React  
- Vite  
- Tailwind CSS  
- @tailwindcss/vite  
- Shadcn/UI

- Canvas API

## Como rodar o projeto localmente

Siga os passos abaixo após clonar o repositório.

### 1. Clonar o repositório
```bash
git clone https://github.com/Arriatea/Portfolio.git
```

### 2. Abrir o terminal

Ctrl + J 

### 3. Entrar dentro da pasta portfolio
```bash
cd portfolio
```
### 4. Instalar as dependências
```bash
npm install
```
### 5. Instalar o plugin do Tailwind para o Vite
```bash
npm install -D @tailwindcss/vite
```
### 6. Executar o projeto
```bash
npm run dev
```
### 7. Abrir no navegador

Basta clicar com neste link, enquanto aperta a tecla Ctrl
```bash
http://localhost:5173
```
